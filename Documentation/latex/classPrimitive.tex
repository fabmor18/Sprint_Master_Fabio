\doxysubsection{Detailed Description}
This class defines a motion primitive used by the algorithm. Each \mbox{\hyperlink{classPrimitive}{Primitive}} is identified by two \mbox{\hyperlink{structnode__data}{node\+\_\+data}} structures, one defining the initial condition under which the primitive can be applied and one defining the final condition reached after the application of the primitive. Each primitive also contains a variable called path\+\_\+offsets that is a vector of int arrays of size 3 where each array represents a checkpoint for the collision checking stored as relative position difference with respect to the starting node. Each primitive also has a cost, expressed as a float and an int id.
